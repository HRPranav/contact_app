Developed a Front end Template of contact app using UI web Technologies such as HTML,CSS,Java script and Backend using PHP and Database used is My sql 

Instructions to open the contact app
1)Install a PHPMyAdmin Database and a open source web server XAMPP
2)open all the below mentioned files in visual studio
3)create a database in PHPMyAdmin such that it contains username,password,secret
4)New users  after getting sign up the  data will be stored in PHPMYAdmin database
5)existing users can login from the portal,after login it is directed to index page
